299
No Repairs Needed

Code 299 is used when, during the detailed inspection, no infractions are found and no follow-up is needed. Use Code 299 to complete the inspection in Geocall Mobile and move on.

No Infractions Found
Inspection reveals full compliance — no issues detected.

The property meets all current standards and requirements during the detailed inspection phase.

Complete in Geocall Mobile
Enter Code 299 to close out the inspection and proceed.

Geocall Mobile will finalize the record, no additional documentation or follow-up scheduling required.

No Follow-Up Required
No district follow-up or further investigation is recommended.

If any follow-up is needed, a different result code must be used instead of 299.

Important Reminder
If you are recommending the district follow up or investigate further, do not use Code 299. Select the appropriate follow-up code instead.
